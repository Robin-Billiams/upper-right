# Sam-Has-Sung
A single service of an entire point of sale to an ecommerce webpage.

![Upper end of site](/image/W7D4.png)


## run npm install
## start mongo on your machine

1. `npm run build`
2. `sudo service mongodb start`
3. `npm run seed`
4. `npm run start`
